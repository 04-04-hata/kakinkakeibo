# 課金家計簿

## サイト概要
ソーシャルゲームに「いつ、いくら」課金したかをまとめるサイトです。

### サイトテーマ
課金で使ったお金の管理

### テーマを選んだ理由
課金で使った額をまとめて、お金の管理に役立てる為

### ターゲットユーザ
ソーシャルゲームで課金をしている人

### 主な利用シーン
ソーシャルゲームで課金した時にいくら使ったかをまとめる
